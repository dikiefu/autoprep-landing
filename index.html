AutoPrep-Angola

Plataforma de Autogestão do Conhecimento para Exames de Acesso ao Ensino Superior

Sobre o Projeto

A AutoPrep-Angola é uma solução tecnológica desenvolvida como parte de um projeto de doutoramento, com o objetivo de auxiliar estudantes angolanos na preparação para exames de acesso ao ensino superior.

Funcionalidades

  • Simulados interativos com correção automática
  • Trilhas de aprendizagem personalizadas
  • Gamificação e conquistas
  • Certificados com QR Code
  • Painéis para professores e gestores
  • Exportação em PDF e Excel
  • Funciona offline e via aplicativo
